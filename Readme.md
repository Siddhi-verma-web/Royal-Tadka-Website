# 👑 Royal Tadka — Fine Indian Dining Website

A premium, fully responsive restaurant website built with pure **HTML, CSS, and JavaScript** — no frameworks, no backend. Combines royal Indian heritage aesthetics with a modern luxury dining experience.

![Made with CSS](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)
![Made with JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat&logo=javascript&logoColor=black)

---

## ✨ Features

- 🎨 Royal Indian-luxury themed UI with mandala/paisley/rangoli motifs
- 🌗 Dark / Light mode toggle
- 🍛 Full interactive menu with category filters (North Indian, South Indian, Chinese, Italian, Mughlai, Street Food, Fast Food, Desserts, Drinks)
- 🥗 Veg-only filter toggle
- 🛒 Shopping cart with GST calculation, delivery charges, promo codes, and live order tracking (all in-browser, no server)
- 📅 Table booking form with confirmation popup
- 🖼️ Masonry-style photo gallery with lightbox
- ⭐ Animated testimonials carousel
- 📊 Animated statistics counters
- ❓ FAQ accordion
- 🔍 Live dish search
- 📱 Fully responsive — mobile, tablet, and desktop

---

## 🛠️ Tech Stack

**Frontend only:** HTML5, CSS3, Vanilla JavaScript, Font Awesome icons, Google Fonts

## 🚀 Getting Started

1. Clone this repository
   ```bash
   git clone https://siddhi-verma-web.github.io/Royal-Tadka-Website/
   ```
2. Open the folder in VS Code
3. Install the **Live Server** extension and click "Go Live" on `index.html`
   (or simply double-click `index.html` to open it directly in your browser)
